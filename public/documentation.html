My movies API

This API allows developers to access and interact with my movie database. It provides endpoints for retrieving movie information, adding new movies, and more.

Request URL Method Response
Get All Movies /movies GET JSON array of all movies
Get Movie by Title /movies/:title GET title (string) None JSON object containing movie details:
{
"title": "The Shawshank Redemption",
"description": "...",
"genre": "Drama",
"director": "Frank Darabont",
"imageUrl": "...",
"featured": false
} 		Returns details of a movie based on its title.
Get Genre by Name /genres/:name GET name (string) None JSON object containing genre details:
{
"name": "Drama",
"description": "..."
} 		Returns the description of a genre based on its name.
Get Director by Name /directors/:name GET name (string) None JSON object containing director details:
{
"name": "Frank Darabont",
"bio": "...",
"birthYear": 1959,
"deathYear": null
} 		Returns details of a director based on their name.
Register New User /users POST None JSON object containing user details:
{
"username": "newuser",
"password": "password123",
"email": "newuser@example.com",
"birthdate": "1990-01-01"
} 		Text: "User registered successfully" Registers a new user.
Update User Info /users/:username PUT username (string) JSON object containing updated user details:
{
"username": "updateduser",
"password": "newpassword456",
"email": "updateduser@example.com",
"birthdate": "1995-05-05"
} 		Text: "User info updated successfully" Updates user information.
Add Movie to Favorites /users/:username/movies/:movieId POST username (string), movieId (string) None Text: "Movie added to favorites" Adds a movie to a user's favorites list.
Remove Movie from Favorites /users/:username/movies/:movieId DELETE username (string), movieId (string) None Text: "Movie removed from favorites" Removes a movie from a user's favorites list.
Deregister User /users/:username DELETE username (string) None Text: "User deregistered successfully" Deregisters a user.
