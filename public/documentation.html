My movies API

This API allows developers to access and interact with my movie database. It provides endpoints for retrieving movie information, adding new movies, and more.

Request URL Method Parameters Request Body Example Response Description
Get All Movies /movies GET None None Response: 
[
  {
    "_id": "65...",
    "Title": "The Shawshank Redemption",
    "Description": "...",
    "Genre": { "Name": "Drama", "Description": "..." },
    "Director": { "Name": "Frank Darabont", "Bio": "..." },
    "Actors": ["Tim Robbins", "Morgan Freeman"],
    "ImagePath": "shawshank.jpg",
    "Featured": true,
    "__v": 0
  },
  // ... more movies
]
Returns a list of all movies.
Get Movie by Title /movies/:title GET Path Parameter: title (string) None Response: 
{
  "_id": "65...",
  "Title": "The Shawshank Redemption",
  "Description": "...",
  "Genre": { "Name": "Drama", "Description": "..." },
  "Director": { "Name": "Frank Darabont", "Bio": "..." },
  "Actors": ["Tim Robbins", "Morgan Freeman"],
  "ImagePath": "shawshank.jpg",
  "Featured": true,
  "__v": 0
}
Returns details of a movie based on its title.
Get Genre by Name /genres/:name GET Path Parameter: name (string) None Response: 
{ "Name": "Drama", "Description": "..." }
Returns the description of a genre based on its name.
Get Director by Name /directors/:name GET Path Parameter: name (string) None Response: 
{ "Name": "Frank Darabont", "Bio": "..." }
Returns details of a director based on their name.
Register New User /users POST None Request Body Example: 
{
  "Username": "newuser",
  "Password": "password123",
  "Email": "newuser@example.com",
  "Birthday": "1990-01-01"
}
Response: 
{
  "_id": "65...",
  "Username": "newuser",
  "Password": "password123",
  "Email": "newuser@example.com",
  "Birthday": "1990-01-01T00:00:00.000Z",
  "FavoriteMovies": [],
  "__v": 0
}
Registers a new user.
Update User Info /users/:username PUT Path Parameter: username (string) Request Body Example: 
{
  "Username": "updateduser",
  "Password": "newpassword456",
  "Email": "updateduser@example.com",
  "Birthday": "1995-05-05"
}
Response: 
{
  "_id": "65...",
  "Username": "updateduser",
  "Password": "newpassword456",
  "Email": "updateduser@example.com",
  "Birthday": "1995-05-05T00:00:00.000Z",
  "FavoriteMovies": [],
  "__v": 0
}
Updates user information.
Add Movie to Favorites /users/:username/movies/:movieId POST Path Parameter: username (string), movieId (string) None Response: 
{
  "_id": "65...",
  "Username": "user",
  "Password": "password",
  "Email": "user@example.com",
  "Birthday": "1990-01-01T00:00:00.000Z",
  "FavoriteMovies": ["65..."],
  "__v": 1
}
Adds a movie to a user's favorites list.
Remove Movie from Favorites /users/:username/movies/:movieId DELETE Path Parameter: username (string), movieId (string) None Response: 
{
  "_id": "65...",
  "Username": "user",
  "Password": "password",
  "Email": "user@example.com",
  "Birthday": "1990-01-01T00:00:00.000Z",
  "FavoriteMovies": [],
  "__v": 1
}
Removes a movie from a user's favorites list.
Deregister User /users/:username DELETE Path Parameter: username (string) None Response: 
{ "message": "user was deleted." }
Deregisters a user.
